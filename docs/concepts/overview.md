# Concepts Overview

MozaiksCore sits at the foundation of the MozaiksAI stack.

## Architecture

MozaiksCore handles:

1. **Authentication & Authorization** – User identity and access control
2. **Subscription Management** – Plan enforcement and billing hooks
3. **Multi-Tenancy** – Isolated app and user contexts

## Integration Points

MozaiksCore integrates with:

- **MozaiksAI Runtime** – Executes declarative workflows
- **MozaiksPay** – Token tracking and usage analytics
