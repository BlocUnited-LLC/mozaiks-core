# Welcome to MozaiksCore

MozaiksCore is the production foundation and agentic backbone for the MozaiksAI ecosystem.

## Overview

MozaiksCore provides:

- **User Management** – Authentication and user lifecycle
- **Subscription Management** – Billing and plan enforcement
- **Agentic Backbone** – Runtime integration and ChatUI transport
